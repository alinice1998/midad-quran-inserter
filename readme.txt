A Gutenberg block plugin for WordPress that allows users to easily search and insert Quranic verses in beautiful Uthmani script via the Kalimat API.

== Description ==

Midad Quran Inserter is a powerful and elegant solution for adding Quranic verses to your WordPress posts and pages. Built on top of the robust Kalimat API, it offers a seamless integration with the Gutenberg block editor.

### Key Features
*   **Gutenberg Native:** Fully integrated as a custom block, providing a smooth editing experience.
*   **Search Flexibility:** Search using Arabic words, Surah names, or numeric formats like `Surah:Ayah` (e.g., `2:255`).
*   **Uthmani Script:** Ensure absolute accuracy and beauty with authentic Uthmani script formatting.
*   **Kalimat API Powered:** Delivers incredibly fast and highly accurate search results.
*   **Easy Insertion:** Insert verses directly into your content with a single click.
*   **Translation Ready:** Fully localized (currently available in English and Arabic).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/midad-quran-inserter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Obtain an API Key from [Kalimat.dev](https://kalimat.dev/).
4. Go to Settings -> Midad Quran and enter your API Key.
5. In your page/post editor, search for the 'Midad Quran Inserter' block and add it to your content.

== Frequently Asked Questions ==

= Does this plugin require an external API? =

Yes, this plugin relies on the Kalimat API (kalimat.dev) to process searches and return Quranic text. You need to obtain a free API key from their website.

= Can I use this with the Classic Editor? =

No, this plugin is designed specifically for the modern WordPress Block Editor (Gutenberg) and will not work with the Classic Editor.
